0. Создать сервер + настроить ssh + подключить домен
---

1. Устанавливаем python3.9 + python3.9-venv, git, nginx, gunicorn
2. Скачиваем код из гитхаба
3. Создаем виртуальное окружение и устанавливаем зависимости
4. Прогоняем миграции
5. Создаем файл конфигурации в sites-available
6. Подключить конфигурацию в sites-enabled --> [Перезапустить Nginx]
7. Настраиваем gunicorn --> [Перезапустить gunicorn]
8. Собираем статические файлы
9. Создаем файл .env с переменными: 

IS_DJANGO_DEBUG_FALSE = y,
DEBUG = False
ALLOWED_HOSTS = ['{SITENAME}']

10. Генерируем и подставляем в .env SECRET_KEY

---
11. Запускаем nginx
12. Запускаем gunicorn
